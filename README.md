Binary Tree Assignment with MVC Web application design.

This is a ASP.NET CORE MVC Web Application.  The FrontEnd is provided as a Website which is rendered on the server(SSR).  When the user inputs some text to the website, the HTML forms
will send a GET request to the backend application.  The Controller class will pass the inputed data to Model classes, namely a Binary Search Tree, to compute the inputs and return some appropitate data to the controller
The controller will call a 'View' and pass in the new data.  The View is HTML using BootStrap for the styling and its HTML structures.  Razor sytax is used to pass the partial views into 
the home page.
